# Video comparison

The Video Comparison tab lets you analyze selected videos side by side. Compare performance metrics, SEO scores, publish timing, and view projections across multiple dimensions.

<Screenshot
  src="https://raw.githubusercontent.com/ytScope/ytscope-content/main/images/academy/en/videos-compare.webp"
  alt="Video Compare tab"
  caption="Compare tab — select videos from the List tab to add them to the comparison."
/>

Click the **Compare** tab on the Videos page. Comparison sections appear once you select at least 2 videos.

---

## Selecting videos

Select the videos you want to compare from the **List** tab. Selected videos automatically appear in the Compare tab.

| Detail | Description |
|--------|-------------|
| **Minimum** | At least **2 videos** are required for comparison. |
| **Maximum** | You can select up to **3 videos** (or more, depending on your plan). |
| **Video counter** | The indicator at the top shows the number of selected videos (e.g., 0/25). |

---

## Comparison sections

Once videos are selected, the following analysis sections appear:

### Benchmark metrics

Displays key metrics for selected videos in a side-by-side table.

| Metric | Description |
|--------|-------------|
| **Average views** | Average view count across the videos. |
| **Median views** | Middle value unaffected by outliers. |
| **Average likes** | Average likes per video. |
| **Average engagement** | Average engagement rate (%). |
| **Daily views** | Average daily views per video. |

### Radar chart

A 7-axis radar chart visually compares each video's strengths and weaknesses:

| Axis | Description |
|------|-------------|
| **Views** | Total view count. |
| **Likes** | Total like count. |
| **Comments** | Total comment count. |
| **Engagement** | Engagement rate (%). |
| **Daily views** | Average daily views. |
| **SEO score** | Search optimization score. |
| **Velocity** | View momentum (velocity). |

The dashed line represents the channel average. Areas extending beyond this line indicate above-average performance.

### Performance chart

View the videos' view trends as a bar or line chart. Use the period selector to change the time range.

### View velocity

Shows each video's view velocity over the last 24 and 48 hours, daily rate, and trend direction.

### SEO comparison

| Metric | Description |
|--------|-------------|
| **SEO score** | Each video's search optimization score (shown with a progress bar). |
| **Tag count** | Number of tags used by each video. |
| **Common tags** | Tags found across all selected videos. |
| **Unique tags** | Tags specific to each video. |
| **Description length** | Description text length comparison. |

### Publish timing

Shows the day and hour each video was published side by side. Evaluate which timing delivered better performance.

### View projection

Displays estimated 30-day view counts with a confidence level for each video.

<Callout type="tip">
Look for dimensions below the channel average on the radar chart. These areas highlight where improvement is needed.
</Callout>

<ProTip>
Compare content published on the same topic in different formats (Shorts vs. long-form). Measure how format affects performance.
</ProTip>
