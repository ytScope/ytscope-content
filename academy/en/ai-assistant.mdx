By the end of this guide, you can ask **Ask AI** from the right screen context, verify a response against measured data, and recognize actions AI Copilot does not take.

## Before you begin

- Select the relevant channel, video, tab, period, and filters before asking.
- Check remaining usage in the panel or under **Settings > AI Credits**.
- Do not enter passwords, API keys, payment data, or confidential client information.
- Do not use AI output as a final financial, legal, or commercial decision.

## Open the panel and establish context

Select **Ask AI** in the lower-right corner. The panel contains suggested questions, a message field, credit status, new-chat controls, and history.

For a reliable workflow:

1. Open the relevant page.
2. Select the channel, video, tab, period, and filters.
3. Choose a suggested question or write one decision question.
4. Verify numeric claims in a visible card or table.
5. Ask which data and period were used when context is unclear.

## What context can AI Copilot use?

| Page | Visible context it may use | Example question |
|---|---|---|
| **Dashboard** | Selected channel, health/weekly cards, and competitor summaries | “What are the two signals I should review this week?” |
| **Studio** | Competitor signals, predictor filters, and generation output from the current page session | “What evidence supports this title suggestion?” |
| **Channels** | Selected channels and comparison context | “Which channel is more consistent in the same period?” |
| **Videos** | Active tab, channel/period filters, and selected videos | “Compare these two videos using the same measures.” |
| **Rankings** | Metric, period, category, and visible list scope | “Explain the scope and bounded conclusion of this ranking.” |

AI Copilot works with context available to you in YTScope. It does not unlock a feature or obtain private YouTube Studio metrics that YTScope does not measure.

## Write a better question

| Weak question | Better question |
|---|---|
| “How is my channel?” | “For the selected Dashboard channel, explain the strongest and weakest visible weekly signals and name the cards.” |
| “What should I make?” | “From Studio's 7-day competitor signals, suggest two topics and state evidence and uncertainty for each.” |
| “Which is good?” | “Compare the two selected videos at similar age and format; state any missing data.” |

Ask for the result, then evidence, then action instead of combining many tasks in one message.

## Verify the response

- Match the channel and period in the answer with the screen.
- Find each number, rank, or percentage in the related card.
- If AI claims causation, ask whether the evidence only shows correlation or a pattern.
- Verify facts and brand voice in any title, description, or strategy suggestion.
- Reject claims about click-through rate, watch time, conversions, or demographics when those values are not measured.

<Callout type="warning">
AI can produce a fluent but incorrect or outdated response. Do not share an unverified claim as fact.
</Callout>

## Credits and chat history

Check credit status before sending. Credit costs and access may change, so this guide does not hard-code an amount.

History reopens earlier conversations. Use **New Chat** when changing topics to avoid mixing contexts. If delete is available, read its confirmation; deleting chat history does not remove channels or product data.

## Privacy and non-actions

AI Copilot:

- responds using your message and available product context;
- does not publish or edit a YouTube video;
- does not add, reassign, or remove a channel;
- does not buy a plan, cancel a subscription, or make a payment;
- should not request an API key and is not a secure vault for secrets.

If a response recommends an action, you complete it on the relevant page.

## Loading, empty, error, and locked states

| State | Next step |
|---|---|
| Response is generating | Wait instead of sending the same message again. |
| Context missing or wrong | Close the panel, open the right page and selections, then start a new chat. |
| Credits unavailable | Check **Settings > AI Credits**. |
| Feature locked | Read the locked panel and check **Subscription**. |
| Timeout or rate limit | Retry later with a narrower question. |
| Empty history | Send the first message to create a conversation. |

## Limits and troubleshooting

- Open the correct page and filters before using phrases such as “this page.”
- Start a new chat when the topic changes.
- If answer and screen disagree, treat the screen as source and report the mismatch.
- Remove personal and client information before sharing AI output.
