# Predictions

The Predictions tab provides forward-looking data for the channel. Use the momentum score to gauge growth direction, churn analysis for risk assessment, and the first 48-hour forecast to evaluate early video performance.

Click the **Predictions** tab on the channel detail page to access this section.

## Momentum score

The Momentum Score summarizes the channel's growth acceleration in a single number. Based on the last 14 days of data, it shows whether the channel is trending up or slowing down.

| Metric | Description |
|--------|-------------|
| **Score** | Value between -100 and +100. Positive values indicate growth momentum, negative values signal deceleration. |
| **Status badge** | **Rising**, **Stable**, or **Declining** — the channel's overall direction. |
| **Subscriber Change (7d)** | Net subscriber change over the last 7 days and percentage difference vs. the previous 7 days. |
| **View Change (7d)** | Total view change over the last 7 days and comparison with the prior period. |

The **Insights** section below the score provides AI-powered short commentary.

<Callout type="tip">
Track the momentum score weekly. If a channel shows a declining trend for two consecutive weeks, review its content strategy.
</Callout>

## Subscriber churn analysis

Churn analysis measures how well the channel retains its subscribers.

| Metric | Description |
|--------|-------------|
| **Churn rate** | Percentage of subscribers lost during the period. |
| **Churn velocity** | Average daily subscriber loss count. |
| **Daily change timeline** | Chart showing subscriber gains and losses by day. |
| **Weekly breakdown** | Week-over-week comparison — is churn accelerating or decelerating. |

<Callout type="note">
Some level of subscriber churn is normal. Viewers naturally lose interest over time. What matters is that the churn rate doesn't exceed the acquisition rate.
</Callout>

## First 48-hour forecast

Analyzes the early performance of newly published videos and generates forecasts for future uploads.

| Element | Description |
|---------|-------------|
| **Early performance summary cards** | Predicted average views, likes, and engagement within the first 48 hours. |
| **Per-video forecast table** | First 48-hour performance prediction for each recent video. |
| **Engagement trajectory** | How engagement rate changes after publication. |

This section helps calibrate your expectations before publishing a new video.

<ProTip>
Compare first 48-hour forecasts with actual results. Videos that outperform the forecast represent the channel's strengths.
</ProTip>

## View velocity decay

Analyzes how quickly view velocity drops after a video is published. Defines two main video categories:

| Category | Description |
|----------|-------------|
| **Viral** | Videos that get high initial views but drop off quickly. |
| **Evergreen** | Videos that continue to receive steady views over a long period. |

| Element | Description |
|---------|-------------|
| **Dominant category** | Whether the channel's videos lean viral or evergreen. |
| **Category distribution chart** | Visual split of videos between viral and evergreen. |
| **Top viral videos** | Videos that captured the highest view momentum right after publishing. |
| **Top evergreen videos** | Videos that continue attracting views long after publication. |

<Callout type="tip">
Channels with a high evergreen ratio draw traffic from search. Viral-heavy channels depend on the suggested videos feed.
</Callout>

## Correlation analysis

Statistically measures the relationships between channel metrics.

| Element | Description |
|---------|-------------|
| **Correlation matrix** | Table showing relationships between metrics (e.g., the link between views and engagement). |
| **Method** | Choose between Pearson or Spearman correlation coefficients. |
| **Sample size** | Number of data points included in the analysis. |

**Interpreting correlation values:**

| Value | Meaning |
|-------|---------|
| **+0.7 to +1.0** | Strong positive relationship — when one metric rises, so does the other. |
| **+0.3 to +0.7** | Moderate positive relationship. |
| **-0.3 to +0.3** | Weak or insignificant relationship. |
| **-0.7 to -0.3** | Moderate negative relationship. |
| **-1.0 to -0.7** | Strong negative relationship — one rises while the other falls. |

<Callout type="note">
Correlation doesn't imply causation. A high correlation between two metrics doesn't mean one causes the other. However, it does provide useful strategy clues.
</Callout>
