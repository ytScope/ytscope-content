By the end of this guide, you can rank tracked channels on one measurement, read your position, and verify scope before sharing a result.

## Before you begin

- Add the channels you want to compare and confirm their roles.
- To see your own position, assign the correct channel as **My Channel**.
- Export actions appear according to your account's current feature access.

## How to reach this screen

Select **Rankings** from the left navigation. The filters update the summary, **Top 3**, **All Rankings**, and user bar together.

## Filters and measurements

| Control | Selection | Reading rule |
|---|---|---|
| **Metric** | **Subscriber Growth**, **Views**, **Likes**, **Engagement Rate**, **Video Count** | Do not equate rank across different metrics. |
| **Period** | The recent-day windows shown on screen | Separate short-term movement from longer-term persistence. |
| Category | Narrows the channel group | Results apply only within the selected category. |
| **Show** | Sets the **Top** channel count | It changes visible list length, not the underlying ranking measure. |
| **Search channels...** | Searches the visible list | Search does not change metric or period. |

**Engagement Rate** is a ratio, while many other options are counts. Do not read first place as “best channel overall.”

## Summary, podium, and list

| Section | What it shows |
|---|---|
| **Total Channels** | Channel count within active filters. |
| **Top Performer** | Leader for the selected metric and period. |
| **Average** | Arithmetic mean; sensitive to outliers. |
| **Median** | Middle ordered value; often more representative of a typical channel. |
| **Top 3** | Leading three when enough channels are available. |
| **All Rankings** | Rank, movement, metric value, and gap to the channel above. |

A trend arrow shows movement from the previous comparable period. When values tie, do not invent a tie-break rule; report the displayed rank and value together.

## Your channel and Your Arena Status

The user bar combines **You**, rank movement, daily velocity, and subscriber gap to the channel above. Catch-up is based on subscriber gap and observed subscriber velocity. If **Views** or **Engagement Rate** is selected, do not mistake it for a forecast of that metric.

AI text under **Your Arena Status** is motivational, not analytical evidence. Return to the list, period, and metric for a numeric conclusion.

## Refresh, sharing, and export

### Refresh

The information tooltip states that data refreshes on a six-hour cadence. Processing can still be delayed, so record the visible last-updated time in a report.

### Share My Rank

| Action | Result |
|---|---|
| **Download Card** | Creates an image from the visible ranking summary. |
| **Copy Link** | Creates a ranking snapshot that can open without signing in. |
| Social sharing | Sends the same public link to the selected platform. |

Treat the share link as public. The audited UI did not expose a revoke control, so do not assume the link can be canceled. Before sharing, consent to exposing the channel name, rank, growth, percentile, and competitor summary.

### CSV and PDF

**Export CSV** and **Export PDF** appear only with current access. Open the file and verify metric, period, channel count, and generation time; the filename alone does not prove scope.

## Practice with Coğrafyanın Kodları

**Verified July 17, 2026.**

1. Set **Metric** to **Subscriber Growth**, then to **Engagement Rate**.
2. Keep the same **Period** and category, and note how the rank for **Coğrafyanın Kodları** changes.
3. Compare **Average** and **Median**; a wide gap suggests checking for an outlier channel.
4. Read the gap to the channel above and the trend arrow.
5. If the user bar appears, interpret subscriber catch-up separately from the selected metric.
6. Open sharing, but do not complete **Copy Link** unless you want a public snapshot.

## Loading, empty, error, and locked states

| State | Next step |
|---|---|
| **No rankings found** | Broaden filters or wait for channel data to be prepared. |
| **No results found** | Clear search and check active filters. |
| No **Top 3** | Fewer than three channels may match; use **All Rankings**. |
| No user bar | Check **My Channel** and whether it matches active filters. |
| Share/export error | Retry; if no file or link exists, do not treat the action as complete. |
| Action hidden or locked | Check current access on **Subscription**. |

## Limits and troubleshooting

- Rankings represent only tracked channels matching active filters.
- Movement does not explain cause; verify it in Channel Analysis and period videos.
- Do not treat a public link as private client material.
- If a share must be removed, contact support because no revoke control was visible in the audited UI.
