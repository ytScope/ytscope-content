# YouTube Metrikleri Rehberi

Doğru metrikleri anlamak, kanalınızı büyütmenin anahtarıdır. Bu rehberde YTScope'ta gördüğünüz tüm metrikleri açıklıyoruz.

## Temel Metrikler

### Abone Sayısı (Subscribers)

Kanalınıza abone olan toplam kullanıcı sayısı.

**Neden önemli?**
- Sadık kitle göstergesi
- Monetizasyon şartı (1000 abone)
- Sosyal kanıt

**Sağlıklı büyüme:**
- Aylık %5-10 büyüme iyi
- Ani düşüşler içerik sorununu gösterir
- Viral videolar geçici spike yaratır

### Görüntülenme (Views)

Videolarınızın toplam izlenme sayısı.

**Sayılma kuralı:**
- 30 saniyeden uzun izlemeler sayılır
- Aynı kişi birden fazla kez sayılabilir
- Bot görüntülemeleri filtrelenir

**Metrik türleri:**
| Metrik | Açıklama |
|--------|----------|
| Toplam Views | Kanal ömrü boyunca |
| Son 28 gün | YouTube Studio varsayılanı |
| Video başına | Tek video performansı |

### İzlenme Süresi (Watch Time)

Videolarınızın toplam izlenme süresi (saat cinsinden).

**Neden kritik?**
- Monetizasyon şartı (4000 saat/yıl)
- Algoritma sıralamasında #1 faktör
- Uzun izlenme = kaliteli içerik

**Formül:**
```
Watch Time = Ortalama İzlenme Süresi × Görüntülenme
```

### Etkileşim Oranı (Engagement Rate)

İzleyicilerin içerikle ne kadar etkileşime girdiği.

**Hesaplama:**
```
Engagement = (Likes + Comments) / Views × 100
```

**Benchmark:**
| Oran | Değerlendirme |
|------|---------------|
| < 1% | Düşük |
| 1-3% | Ortalama |
| 3-6% | İyi |
| > 6% | Mükemmel |

## İleri Seviye Metrikler

### Tıklama Oranı (CTR)

Gösterimden tıklamaya dönüşüm oranı.

**Hesaplama:**
```
CTR = Tıklama / Gösterim × 100
```

**Etkileyen faktörler:**
- Thumbnail kalitesi
- Başlık çekiciliği
- Video konusu

**Ortalamalar:**
- Yeni kanallar: %2-5
- Kurulu kanallar: %5-10
- Viral içerik: %10+

### Ortalama İzlenme Oranı (AVD%)

Videonun ortalama yüzde kaçının izlendiği.

**Hesaplama:**
```
AVD% = Ortalama İzlenme Süresi / Video Süresi × 100
```

**Hedefler:**
- Shorts: %80+
- 10 dk video: %50+
- 20+ dk video: %40+

### Abone Kazanımı (Subs Gained)

Her videodan kazanılan net abone sayısı.

**Formül:**
```
Net Abone = Yeni Abone - Abone Kaybı
```

## YTScope Özel Metrikleri

### Performans Skoru

YTScope'un geliştirdiği bileşik performans metriği.

**Hesaplama faktörleri:**
- Görüntülenme hızı
- Etkileşim oranı
- Abone dönüşümü
- İzlenme süresi

**Skor aralığı:** 0-100

### Trend Skoru

İçeriğin viral olma potansiyeli.

**Faktörler:**
- İlk 24 saat performansı
- Paylaşım oranı
- Yorum aktivitesi

### Rakip Kıyaslama

Metriklerinizi rakiplerinizle karşılaştırma.

**Karşılaştırılan metrikler:**
- Ortalama görüntülenme
- Etkileşim oranı
- Yükleme sıklığı
- Büyüme hızı
